Manipular o CSS

Para fazer modificações usando o CSS, por aqui mesmo.
Usa o mesmo contesto, que você usa no CSS.
Exemplo, vai pegar na posição 2, da tag "h1",vai fazer as seguintes modificações:
Muda o estilo da cor, muda o tamanho da fonte, muda até o texto que estava na tag.


Exemplo de código:

"script" >> abre a tag

//funçao.
function acao (){


_________//obter o elemento h1 = pega o h1, na posição "2", o que tem dentro dele e salva na var "texto".
_________var texto =document.getElementsByTagName("h1")[2];


_________//CSS = muda "estilo da cor", do que tem na var "texto"
_________texto.style.color = "#ffAa21";


_________//CSS = muda o "tamanho da fonte", do que tem na var "texto"
_________texto.style.fontSize = "5rem";


_______//pega o "Curso....", E salva na variável "texto", com as modificações adicionadas "CSS".
_______texto.innerHTML = "Curso de JavaScript";


}
