Estrutura base do JavaScript

Essa é a estrutura base do JS.
"script" abre, "/script" fecha, e entre as tags adiciona o código.
É bom colocar dentro do "head", no html, ela faz aparecer uma "janelinha" na página.

Caixa de mensagem, é essa janela, para dá aviso ao usuário, orienta ele no site.
