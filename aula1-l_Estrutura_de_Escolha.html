Estrutura de escolha "SWITCH"

Para executar ações, quando você tem várias opções.
Como um menu, várias coisas, que cada uma dá uma respota diferente.
Não usa os condicionais relacionais (maior q,menor q,igual q...).
Pode ser usado com condicionais que usam "if".
No exemplo:
Menu de restaurante.


Exemplo de código:
"script" >> abre a tag

//variável
var codigo = 1;


//escolha (de acordo com o código)
switch(codigo) {

//caso seja 1; vai exibir essa mensagem.
____case 1:
_______alert("Pizza");
//break = parar
____break:

//caso seja 2; vai exibir essa mensagem.
____case 1:
_______alert("Suco");
____break:

//caso seja 3; vai exibir essa mensagem.
____case 3:
_______alert("Macarrão");
____break:

____default:
_________alert("Código inválido.");
