Condicionais Simples no JS

Usados para fazer verificações de acordo com as condições.
Passa uma regra, uma comparação e ela diz a respota de acordo com a condição.
No exemplo, para verificar se a pessoa pode ou não tirar CNH, que só pode aparti dos 18 anos.


Exemplo de código:
"script" >> abre a tag

//variável
var idade = 17 ;


//se (idade for maior ou igual a 18)
if(idade >= 18){

//exibi essa mensagem
alert("Aprovado.")

//senão
} else{
//exibi essa mensagem
alert("Reprovado.")
}
