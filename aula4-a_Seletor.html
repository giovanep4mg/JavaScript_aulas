Seletor através de Classe

Para selecionar o conteúdo de alguma tag, pelo sua classe.
Você especifica qual é a tag, que quer modificar.
Pode ser tags diferentes, com o mesmo nome de classe
Para selecionar qual é a posição, usa "[?]", começando com zero.

Como é o código na tag, para ser selecionada:

dentro da tag "h1 class="texto"> Titulo em h1
dentro da tag "h2 class="texto"> Titulo em h2
dentro da tag "p class="texto"> Titulo em Parágrafo
