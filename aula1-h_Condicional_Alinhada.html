Condicionais Aninhada no JS

Usados para fazer verificações de acordo com as condições.
Indicado quando você tem várias respostas.
No exemplo: Escola, media de um aluno.


Exemplo de código:
"script" >> abre a tag

//variável
var media = 8 ;


------Condicional Aninhada--- //se ( media for igual a 10 )
----if ( media == 10 ) {

//exibi essa mensagem
------------alert("Parabéns.")

//senão , se ( media for maior ou igual a 9)
} else if ( media >= 9 ) {
//exibi essa mensagem
------------alert("Ótimo.")

//senão , se ( media for maior ou igual a 8)
} else if ( media >= 8 ) {
//exibi essa mensagem
------------alert("Bom.")

//senão , se ( media for maior ou igual a 7)
} else if ( media >= 7 ) {
//exibi essa mensagem
------------alert("Na média.")

//senão , se ( media for maior ou igual a 5)
} else if ( media >= 5 ) {
//exibi essa mensagem
------------alert("Em análise.")
} else {
------------alert("Reprovado.")
}
