Vetores

Vetor = são várias informações dentro de uma variável,um banco de dados temporário.
Para mostra um item dentro do vetor, usa o "[]", passando o numero de qual você quer.
A ordem dentro do vetor começa com zero "0", ou seja,"HTML" é "0", "CSS" é "1" ...


Exemplo de código:

"script" >> abre a tag

//variável
var cursos = ["HTML", "CSS", "JAVASCRIPT", ];

//exibirá o curso "0", que é o HTML
alert(cursos[0])

//exibirá o curso "2", que é o JAVASCRIPT
alert(cursos[2]);


"/script" > fecha a tag
