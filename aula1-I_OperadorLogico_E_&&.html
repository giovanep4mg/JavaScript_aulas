Operadores "E" ( && )

Usados para comparar duas ou mais informações.
Trabalha junto com condicional, e as duas tem que estarem certas para ser válidada.
No exemplo:
Escola, comparando a média do aluno com as suas faltas, para poder passar.


Exemplo de código:
"script" >> abre a tag

//variável var media = 10; var faltas = 5;

//se (media for maior ou igual a 7) E (faltas for menor ou igual a 15)
if((media >= 7) && (faltas <= 15)){
_______alert("Aprovado!");
}else{
_______alert("Reprovado!");
}

"/script" > fecha a tag
