Operadores Relacionais no JS

Usados para fazer verificações:
Se é maior ou menor,se é igual ou diferente.
Se é maior ou igual, se é menor ou igual.
Vai retornar Verdadeiro "true" ou Falso "false".
* Desativei alguns no código fonte.


Exemplo de código:
"script" >> abre a tag

//Maior
window.alert(5 > 4);
//5 é maior que 4, vai dizer se é ou não.


//igual >> usa dois simbolos de iguais "=="
window.alert(5 == 4);
//5 é igual a 4, vai dizer se é ou não.


//diferente > usa esses simbolos "!="
window.alert(5 != 4);
//5 é diferente de 4, vai dizer se é ou não.

"/script" > fecha a tag
