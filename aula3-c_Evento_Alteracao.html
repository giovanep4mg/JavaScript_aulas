Evento Alteraração

* onchange *
Cria uma caixa de seleção, com algumas opções.
Vai capturar o que foi selecionado, e mostra ao usuário na "janelinha".


Exemplo de código:

"script" >> abre a tag

//criar a função mensagem,quando clicar vai chamar essa funçao.
function mensagem (){


//pega o valor dentro do id "cursos", e salva na variável "curso"
var curso = document.getElementById("cursos").value;


//exibir o curso selecionado, com um texto "Seu curso..."
alert("Seu curso selecionado foi: "+curso);

}


"/script" > fecha a tag

Será exibido uma caixinha de seleção, abaixo:
Que ao selecionar o curso desejado, irá abrir a "janelinha", mostrando qual foi o curso selecionado.
