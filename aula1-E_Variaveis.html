Variáveis no JS

São palavras para armazerna, tipos de dados.
Como foi exibidos na "janelinha", antes desse texto.
Não utilize caracteres especiais.
Não começa com números, pode colocar depois.
Não coloque as palavras separadas, deve ser assim: "tipoAssim","casaNumero"...
Para numeros decimais, utilize o ponto ".", e não a vírgula ",".


Exemplo de código:
"script" >> abre a tag

"//" isso é um comentário > Variáveis .
var nome = "Giovani";
var idade = 28;
var altura = 1.67;
var estuda = true;


Para ser exibido na página, na "janelinha":
//para exibir na tela.
window.alert("Seu nome: "+nome);
window.alert("Sua idade: "+idade);
window.alert("Sua altura é: "+altura);
window.alert("Você estuda: "+estuda);
